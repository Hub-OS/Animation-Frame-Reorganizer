Animation Frame Reorganizer
GitHub

A utility for moving animation frames within BoomSheets files.

Not affiliated with BoomSheets.

Usage:

Load an animation file by dropping a .animation and image file on the page.

WASD / Arrows: Moves selected frames.
Shift + WASD / Arrows: Stretches the space between selected frames.
Drag from a blank space with the mouse to select multiple frames.
Drag a frame with the mouse to move selected frames.
